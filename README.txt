
# Adjective vs. Adverb Identifier

This is a simple Streamlit app that identifies adjectives and adverbs in a sentence. 
It is designed as an educational tool for students learning English grammar.

## Features
- Enter a sentence in English.
- Detects adjectives and adverbs using the spaCy NLP library.
- Displays results separately for easy understanding.

## Requirements
- Python 3.8+
- Streamlit
- spaCy
- English language model for spaCy (`en_core_web_sm`)
